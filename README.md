# FireworkTrailColors
A Minecraft Paper Plugin for 1.18.2

Commands:
Shows help page:
```
/ftc help
```
Toggles firework trails on and off
```
/ftc
```
Changes firework trail color:
```
/ftc 0-255 0-255 0-255
```
Example: `/ftc 255 105 180` for pink
